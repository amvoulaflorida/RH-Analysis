# RH-Analysis

## Contexte du projet
Le département RH souhaite **centraliser et analyser ses données** afin d’améliorer la gestion des ressources humaines et faciliter la prise de décision stratégique.

## Objectif du projet
Créer un **tableau de bord interactif avec Microsoft Power BI** pour analyser les principaux indicateurs RH.

## But du projet
Aider les responsables RH à :
- Comprendre les tendances dans les données des employés
- Identifier les problèmes tels que le turnover ou l’absentéisme
- Améliorer les stratégies de recrutement et de gestion des talents
- Prendre des décisions basées sur les données

## Données disponibles
- **Données des employés** : fichier Excel `Hr_dataset.xlsx`

## Outils utilisés
- **Microsoft Excel** : préparation et nettoyage des données
- **SQL** : extraction et manipulation des données
- **Microsoft Power BI** : création du tableau de bord interactif

## Architecture du projet
Les données, initialement stockées dans un fichier Excel, ont été **importées dans une base de données** pour permettre une analyse plus efficace avec SQL.  
Les résultats ont ensuite été visualisés à l’aide de **Microsoft Power BI** pour créer un tableau de bord interactif et dynamique.

## Dashboard 1 – Vue d’ensemble des employés
Permet de **comprendre la structure de l’entreprise et la gestion des employés**.

### Questions métier
- Quels départements ont le plus d’employés ?
- Quels postes ont les salaires les plus élevés ?
- Quel est l’âge moyen des employés ?
- Quelle est la répartition hommes/femmes dans l’entreprise ?
- Quels départements ont les employés les plus expérimentés ?
